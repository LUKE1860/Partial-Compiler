# INSTALL ⚙️
## 1. INSTALL [MESON](https://mesonbuild.com/SimpleStart.html#installing-meson)

## 2. AFTER INSTALLATION INVOKE COMMAND IN THE COMMAND PROMPT
```
 meson setup --wipe --vsenv  builddir
```
## 3. MOVE TXT FILES TO BUILDDIR
```
move function.txt /builddir
```
## 4. CD INTO BUILDDIR
```
cd builddir
```
## 5. INVOKE NINJA
```
ninja
```
## 6. INVOKE EXECUTABLE
```
./partial_compiler.exe
```
